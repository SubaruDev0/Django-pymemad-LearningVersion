¿Qué es un módulo?

Los módulos definen las áreas funcionales del sistema y los permisos disponibles para cada una.

Acciones Estándar
  • Ver: Permite visualizar registros
  • Crear: Permite crear nuevos registros
  • Modificar: Permite editar registros existentes
  • Eliminar: Permite eliminar registros
  • Exportar: Permite exportar datos
  • Importar: Permite importar datos
Jerarquía

Los módulos pueden tener submódulos para organizar mejor las funcionalidades.
